Mari mencari nilai (g o f)

Diketahui himpunan A = {0, 1, 2, ...}, himpunan
B = {..., −2, −1, 0, 1, 2, ...}, dan himpunan C = {1, 2, 3, ...}.

Terdapat fungsi f : A→B dan a∈A, b∈B dengan f : a→b = f(a) = a2,
serta g : B→C dan b∈B, c∈C dengan g : b→c = g(b) = 2b + 2.

Df = A Rf = {0, 1, 4, 9, 16, ...}
Dg = B Rg = {2, 4, 10, 20, ...}

Masukkan nilai anggota himpunan A pada kotak yang disediakan di bawah ini. Lalu, klik tombol "=" untuk mengetahui hasilnya.
